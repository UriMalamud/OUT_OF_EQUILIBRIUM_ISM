We provides two user friendly interactive HTML files with which to visually inspect the simulation results at a resolution of 1024^3, using a modified version of the RAMSES code.
Simulation details are fully described in our publication, titled: "MULTIPHASE TURBULENCE AS THE ORIGIN OF OH+, H2O+ AND H+ COLUMN DENSITY SCATTER IN THE LOCAL ISM"

The file contents are self explanatory.
- The 1st file includes 6 panels with information regadring out of equilibrium H2 and electrons, including temperature and total density in order to provide context.
- The 2nd file includes 4 panels with information regarding trace molecule abundances, including total density in order to provide context.
